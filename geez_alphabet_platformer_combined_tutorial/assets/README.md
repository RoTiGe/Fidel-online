# Assets Folder

This folder contains visual aids for the Geez Alphabet Platformer game.

## Image Sources
Images are loaded dynamically from Unsplash API to provide visual context for each word being learned.

## Fallback Images
If you want to use local images instead, place them in this folder with the following naming convention:
- breakfast.jpg
- hello.jpg
- world.jpg
- book.jpg
- friend.jpg
- water.jpg
- sun.jpg
- moon.jpg
- tree.jpg
- flower.jpg
- lunch.jpg
- dinner.jpg
- mother.jpg
- father.jpg
- sister.jpg
- brother.jpg

The game will automatically detect and use local images if available, otherwise it will load from Unsplash.
